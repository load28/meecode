# MeeCode

Claude Code CLI를 위한 데스크탑 GUI. CLI의 능력을 그대로 두고, 더 잘 읽고 더 잘 따라갈 수 있는 화면을 얹는다.

---

## 왜 만들었나

Claude Code는 강력합니다. 하지만 강력함이 곧 좋은 사용자 경험을 의미하지는 않습니다.

긴 답변과 연쇄적인 도구 호출이 터미널을 빠르게 흘러가면, 지금 무엇을 보고 있는지, 어디까지 진행됐는지, 방금 수정된 파일이 어디였는지 따라가기가 어렵습니다. 마크다운은 평문으로 흩어지고, 파일 경로는 클릭이 안 되고, 진행 중인 작업과 끝난 작업의 경계가 모호합니다.

CLI가 잘하는 일을 다르게 보여줄 화면이 필요할 뿐입니다. **MeeCode는 그 화면입니다.**

---

## MeeCode가 아닌 것

- **Claude Code의 대체재가 아닙니다.** 내부에서 Claude Code CLI를 그대로 실행합니다. 모델 호출, 에이전트 루프, 세션 관리, 슬래시 커맨드, 스킬, 권한 모델 — 전부 CLI의 것입니다.
- **별도의 AI 에이전트가 아닙니다.** 자체 추론도, 자체 도구도, 자체 메모리도 없습니다.
- **새로운 워크플로우를 강제하지 않습니다.** 익숙한 명령과 흐름이 그대로 동작합니다.

목표는 단 하나입니다. **Claude Code를 잘 활용할 수 있는 화면을 제공하는 것.**

## 주요 기능

### 질답 카드 + 분할 패널
왼쪽에는 압축된 질답 카드 목록, 오른쪽에는 펼친 전체 답변. 카드 하나가 한 질문에 대응하며, 클릭하면 오른쪽에서 통째로 펼쳐집니다. 카드 안 긴 답변은 "더 보기"로 토글하고, 스트리밍 도중에도 양쪽이 동기 갱신됩니다.

### 라이브 마크다운 스트리밍
응답이 도착하는 동안 코드블록·표·인용·헤더가 즉시 렌더됩니다. 완료 후에 평문에서 마크다운으로 점프하거나 글자 크기가 변하는 일이 없습니다. 부드러운 타자기 효과로 토큰 청크가 튀어 보이지 않게 합니다.

### Claude가 무엇을 하고 있는지
한 줄짜리 진행 인디케이터가 thinking·도구 호출·Hook·Task subagent 활동을 같은 자리에서 보여줍니다. 인터럽트(ESC) 시에는 진행 중이던 질답 카드에 "중단됨" 마커가 그대로 남아 흐름이 끊기지 않습니다.

### 파일 경로 = 클릭 링크
`Read`·`Edit`·`Write`·`MultiEdit`·`NotebookEdit` 등 파일을 다루는 도구 호출의 경로는 클릭 가능한 링크입니다. 클릭하면 코드 패널에서 해당 파일이 열리고, 패널은 떼어내 별도 창으로 띄울 수 있습니다.

### 도구 승인 카드
Claude가 명령을 실행하기 전, 무엇을 어떤 인자로 실행할지 카드로 먼저 보여줍니다. 그 자리에서 허용/거부를 선택하고, 인자가 마음에 들지 않으면 수정해서 보냅니다. 권한 모델은 Claude Code의 것을 그대로 따릅니다.

### Sticky 스크롤
스트리밍이 길어지면 자동으로 가장 아래에 붙어 따라가고, 위로 올려 이전 내용을 읽고 있을 땐 멈춥니다. `ResizeObserver` 기반이라 폴링 비용이 없습니다.

### 핀 & 위키 자동 정리
의미 있는 답변·코멘트는 핀으로 모아두고, 모인 핀들을 Claude가 프로젝트 위키 마크다운으로 묶어 줍니다. CLI 세션을 닫아도 정리된 지식은 프로젝트 폴더에 남습니다.

### Claude CLI 자동 탐색
첫 실행 시 PATH와 표준 설치 위치에서 Claude CLI를 자동으로 찾아 설정합니다. 못 찾으면 설정 패널에서 경로를 직접 지정할 때까지 입력창이 비활성화되어 잘못된 상태로 보내는 일이 없게 합니다.

## 시작하기

### 필요한 것

- Node.js 18+
- Rust toolchain (Tauri 2.x 빌드용)
- Claude Code CLI

### 개발

```bash
npm install
npm run tauri dev
```

### 빌드

```bash
npm run tauri build
```

빌드된 앱은 `src-tauri/target/release/bundle/` 아래에 있습니다.

### 테스트

```bash
npm run test
```

---

## 기술 스택

- **Tauri 2.x** — Rust 백엔드 + 웹뷰 프론트엔드
- **React 18 + TypeScript** — UI 레이어
- **marked + DOMPurify + Prism** — 마크다운/코드 하이라이트
- **react-resizable-panels** — 좌/우 패널 분할

Claude Code CLI는 자식 프로세스로 실행되며, `--include-partial-messages` 모드로 받은 JSONL 스트림을 프론트 reducer가 세그먼트 단위로 누적합니다.
